Справка

Регистрация

  1. Нажмите кнопку "Зарегистрироваться" в правом верхнем углу
  2. Заполните поля в открывшемся окне
  3. Нажмите кнопку "Регистрация"
  4. Готово!

Вход в личный кабинет

  1. Нажмите кнопку "Войти в личный кабинет" в правом верхнем углу
  2. Заполните поля в открывшемся окне
  3. Нажмите кнопку "Войти"
  4. Готово!

Выход из личного кабинета

  1. Нажмите кнопку "Выйти" в правом верхнем углу
  2. Готово!

Просмотр пользовательской документации

  1. Нажмите кнопку "Справка" в правом верхнем углу
  2. Готово!

Просмотр личного кабинета

  1. Нажмите кнопку "Личный кабинет" в правом верхнем углу
  2. Готово!
